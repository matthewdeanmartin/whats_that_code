# Contributing

The full contributing guide lives in the repository at [`CONTRIBUTING.md`](https://github.com/matthewdeanmartin/whats_that_code/blob/main/CONTRIBUTING.md).

## Quick start

```bash
git clone https://github.com/matthewdeanmartin/whats_that_code.git
cd whats_that_code
uv sync --all-extras
uv run make check-ci
```

See [CONTRIBUTING.md](https://github.com/matthewdeanmartin/whats_that_code/blob/main/CONTRIBUTING.md) for the full guide including how to add new language support, run individual quality gates, and submit a pull request.
